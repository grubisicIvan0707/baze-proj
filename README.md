Projekat: Baza podataka za prodavnicu sportske opreme

Ovaj rad predstavlja model baze podataka izrađen za simulaciju rada prodavnice sportske opreme. Obuhvata:

- Opis poslovnog domena
- ER dijagram (PMOV)
- Relacioni model
- SQL skripte za kreiranje tabela i unošenje podataka
- Kratka analiza i zaključak

Tehnologije korišćene:
- SQL (T-SQL sintaksa)
- Relacione baze podataka
